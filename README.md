## 🛠️ Technology Stack

### Core Tools
- **Languages:** Python 3.8+  
- **Version Control:** Git, GitHub  
- **Development:** Jupyter, VS Code, GitHub Codespaces

### Data & Analytics
- **Data Processing:** Pandas, NumPy  
- **Visualization:** Matplotlib, Plotly, Seaborn
